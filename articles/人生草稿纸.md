她问：「你有房吗？」  
我答：「有……有 GitHub 仓库」

## 五、写给此刻的自己，也写给明天的我

嘿，  
我知道你现在把社交软件装了又卸，  
把「在吗」两个字打了又删，  
在 B 站看「自然卷打理教程」，  
在 GitHub 刷「面试题大全」，  
标签页里还躺着一个「如何优雅地单身」。  

没关系的。  
真的没关系的。  
宇宙不会因为你少谈一次恋爱就坍缩，  
也不会因为你多刷一道算法题就爆炸。  

你可以继续把头发剪成最短，  
省掉每天早上和卷毛的搏斗；  
至少它们每晚都准时打卡；  
你可以继续把「喜欢」两个字存草稿，  
直到某天它自己跳出来按回车。  

也允许你偶尔崩溃——  
在凌晨两点对着显示器骂娘，  
在雨天地铁口羡慕牵手的情侣，  
在母亲问「啥时候带女朋友回家」时，  
用「还年轻」当挡箭牌，  

但请记住：  
**人生的进度条不是线性的，**  
**有人 30 岁才初恋，有人 40 岁才转行，**  
**有人一辈子单身却养了三只猫，**  
**也有人 25 岁结婚 35 岁离婚再去跳伞。**  

你不必在今天晚上就选好 A 或 B，  
你可以先关掉电脑，  
去楼下买一串 3 块钱的炸年糕，  
让辣酱沾满指尖，  
然后抬头看看——  
月亮仍挂在分叉口上空，  
它不催你，也不笑你，  
它只是亮着，  
像一盏等你先迈出脚的路灯。  

走吧，  
哪怕只是先去把头发洗干净，  
也是向宇宙发射了  
「我还在意自己」的信号。
